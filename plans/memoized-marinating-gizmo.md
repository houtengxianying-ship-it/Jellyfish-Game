# 金色クラゲ同士の衝突設定変更

## Context
金色（エチゼンクラゲ、type 11）同士がぶつかると、2つが消えて1つになる挙動になっていた。ユーザーの要望で、金色同士はくっつかないように設定したい。

## アイデア比較

| アイデア | メリット | デメリット |
|---------|---------|-----------|
| **A. バウンスする（推奨）** | 他のタイプとの一貫性がある。物理的に自然。 | 実装簡単 |
| B. 何も起きない | シンプル | 重なり合うのが不自然 |
| C. 特別エフェクトのみ | 派手で楽しい | 実装が複雑 |

## 推奨アプローチ: A. バウンスする

違うタイプ同士と同じように、金色同士も弾き合うようにします。

### 変更ファイル
- `js/physics.js`

### 変更内容

```javascript
// 現在: 同じタイプなら合体
if (bodyA.jellyfishType !== bodyB.jellyfishType) {
    // バウンス処理
    ...
    continue;
}

// 変更後: 同じタイプ かつ type < 11 なら合体
if (bodyA.jellyfishType !== bodyB.jellyfishType || bodyA.jellyfishType === 11) {
    // バウンス処理
    ...
    continue;
}
```

条件式に `|| bodyA.jellyfishType === 11` を追加するだけで、金色同士はバウンスするようになります。
