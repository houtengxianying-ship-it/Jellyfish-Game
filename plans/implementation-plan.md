# クラゲゲーム（Jellyfish Drop & Merge Game）実装プラン

## Context

スイカゲームのクラゲ版を新規作成する。深海をテーマにした美しいビジュアルで、11段階のクラゲを落下・合体させるパズルゲーム。プロジェクトは `/Users/5vhgoh/クラゲゲーム/` に空の状態で存在しており、ゼロから実装する。

参考スクリーンショット: 深海の青い背景に光線が差し込み、透明なバブルに包まれた色とりどりのクラゲが11種類並んでいる。

## 技術スタック

- **HTML5 Canvas** — 全描画を1つのCanvasで処理
- **Matter.js** (CDN) — 物理エンジン（重力、衝突検出、剛体シミュレーション）
- **純粋JavaScript** — ビルドツール不要
- **クラゲはCanvas 2D APIで手描き** — 画像アセット不要

## ゲームメカニクス

- **Canvas**: 420x640px
- **コンテナ**: 内幅390px、左右と底に壁（厚さ30px）
- **重力**: `engine.gravity.y = 1.5`（水中感のある落下速度）
- **物理パラメータ**: `restitution: 0.3`（柔らか弾み）、`friction: 0.05`、`frictionAir: 0.02`
- **合体**: `collisionStart`イベントで同タイプ検出 → 両方削除 → 中間点に次段階を生成
- **重複処理防止**: `isRemoved`フラグで同フレーム内の二重処理を防ぐ
- **ゲームオーバー**: 危険ラインを2秒以上超過で発動
- **ドロップ間隔**: 500msクールダウン

## 検証方法

1. `index.html` をブラウザで直接開く
2. クラゲを落下させ、同種を衝突させて合体を確認
3. スコアが正しく加算されるか確認
4. 段階11のクラゲまで合体できるか確認
5. 危険ラインを超えてゲームオーバーが発動するか確認
6. Play Againで正常にリスタートされるか確認
7. モバイルブラウザでタッチ操作が動作するか確認
